# 🚀 AI-Based Customer Churn Prediction Platform

An end-to-end AI-powered churn prediction system that integrates behavioral, support, billing, and usage data to identify customers at risk of discontinuing service.

---

## 📌 Overview

This platform enables organizations to:

- Predict customer churn using machine learning
- Assign churn risk scores
- Calculate revenue at risk
- Perform bulk scoring via CSV upload
- Provide actionable insights for retention strategies
- Visualize churn analytics via modern dashboard UI

---

## 🧠 Key Features

- Multi-source data integration
- Feature engineering pipeline
- XGBoost churn prediction model
- SHAP-based explainability
- Real-time REST API inference
- Bulk CSV scoring endpoint
- Revenue impact forecasting
- Modern minimal dashboard (Next.js + Tailwind)
